## v1.0.0 2025-05-24

- Booth にて販売開始

## v0.2.2 2025-05-21

- メガネに以下のシェイプキーを追加
  - Temple_Long: つるを長くする
  - Temple_Short: つるを短くする
  - Temple_Open: つるを開く
- FBX に同梱されているマテリアル名を単純なものに変更

## v0.2.1 2025-05-16

- ExMenu の `PB OFF` を有効化しても PB が無効にならない不具合を修正
  - v0.2.0 の GameObject のリネームにより発生していました。
- PB 使用によるメガネ表示中に ExMenu の `PB OFF` を有効化すると、メガネが表示中のまま PB が無効化される不具合を修正
- ExMenu の `Contact OFF` を `Moving OFF` にリネーム
  - 使用パラメータもリネームしましたが、動作に変化はありません。
- ExMenu の全項目がアバター切り替え後も保持されるように変更
- `OFF（非表示）` で MAObjectToggle ではなく アニメーションを使用するように変更
- 内部の変更: Animation Controller 及び Animation Clip の可読性向上

## v0.2.0 2025-05-14

- ディレクトリ名を `HiroMelon` から `Suikompany` に変更
- メガネの3Dモデルを同梱
- 複数使用した際、既に持ってる手に持たないようにパラメータを調整
  - 同時に複数の持ち判定に触れている状態から同時につかんでしまう問題は対策できていません
- 手や頭に移動する際、時間をかけて移動するように変更
- 着用判定を小さめに調整
- `GrabbableEyewear/Contact/LocalEyewearOnHeadContactRcv` を `GrabbableEyewear` 直下に移動
  - これにより `GrabbableEyewear/Contact` を削除
- `BoneProxy` コンポーネント用の GameObject の名前に接頭辞として `Proxy` を追加

## v0.1.3 2025-04-29

- 導入方法などの記述をウェブサイトに移行
- 本ギミックを複数導入してもパラメータが競合しないように
- ExMenu に以下の項目を追加
  - `PB OFF`: PhysBone を無効化
  - `Contact OFF`: Contactによる持ち替えを無効化
